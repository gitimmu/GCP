cloud related storage registry will maintained -- GCR

pull -- pull image

push -- push image to registry

start/stop -- > container start or stop

Image -- server libraries+web app libraries + app code + File system mounting etc
container -- > once we start image it is called as container

Dockerfile is used to buils images in docker.(https://www.tutorialspoint.com/docker/docker_containers.htm)

Dockerfile:
FROM centos:7
MAINTAINER "GCP developers"
RUN yum -y install epel-release
RUN yum -y install nginx
ADD index.html /usr/share/nginx/html/index.html
ADD image.jpeg /usr/share/nginx/html
EXPOSE 80/tcp
CMD ["nginx", "-g daemon off;"]

command: docker build -t docker.io/nginx:v3 /home/uidapp/
